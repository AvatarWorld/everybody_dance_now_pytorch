# everybody_dance_now_pytorch
A PyTorch implementation of ["Everybody Dance Now"](https://arxiv.org/abs/1808.07371) from Berkeley AI lab.

### Environment
Ubuntu     18.04 (But 16.04 should be fine too)
Python     3.6
CUDA       9.0.176
PyTorch    0.4.1post2
